# Heartbeat Tasks

Run these on each heartbeat poll:

## 1. Create Today's Memory File
If today's memory file doesn't exist in the workspace memory folder, create it:
# YYYY-MM-DD Session Notes

## Key Discussions

## Decisions Made

## Open Items

## Learnings

## 2. Append Major Decisions/Findings
During the session, append important items to today's memory file:
- New technical findings
- Customer decisions
- Configuration changes
- Project milestones

Format:
### HH:MM - Topic
- Decision/Finding: ...
- Impact: ...

## 3. Curate to Memory.md
At end of session (or periodically), promote key items to the main memory file:
- Permanent preferences
- Company info
- Credentials/locations
- Important learnings that should persist

## Priority
1. First heartbeat: ensure today's memory file exists
2. During session: append significant events as they happen
3. End of session: curate best items to main Memory.md
